TM02: Přidání úkolu s platným vstupem
Popis: Ověření, zda lze přidat nový úkol, pokud je zadán platný vstup - název ani popis úkolu není prázdný.
Vstupní podmínky: 
	1. Program Task manager je spuštěn a zobrazuje hlavní menu s nabídkou funkcí (1. Přidat úkol, 2. Zobrazit úkoly, 3. Smazat úkol a 4. Ukončit program).
Kroky testu:
	1. Vyber funkci "1. Přidat úkol" stisknutím čísla 1 na klávesnici a potvrď výběr klávesou Enter.
	2. Zadej platný název úkolu a potvrď klávesou Enter.
	3. Zadej platný popis úkolu a potvrď klávesou Enter.
Očekávaný výsledek: 
	1. Po potvrzení výběru funkce pro přidání úkolu program zavolá funkci add_task() a zobrazí se výzva pro zadání názvu úkolu.
	2. Platný název úkolu je zadán a po potvrzení klávesou Enter se zobrazí výzva pro zadání popisu úkolu.
	3. Platný popis úkolu je zadán a po potvrzení klávesou Enter se zobrazí informace o přidání nového úkolu: Úkol 'název_úkolu - popis_úkolu' byl přidán.
Výsledek testu: 
	1. Výběr funkce pro přidání úkolu je potvrzen a je zobrazena výzva pro zadání názvu úkolu.
	2. Platný název úkolu "Koupit chleba" je zadán a potvrzen klávesou Enter. Je zobrazena výzva pro zadání popisu úkolu.
	3. Platný popis úkolu "celozrnný" je zadán a potvrzen klávesou Enter. Informace o úspěšném přidání nového úkolu je zobrazena: Úkol 'Koupit chleba - celozrnný' byl přidán.
Stav: Pass

TM07: Smazání úkolu
Popis: Ověření, zda je možné smazat existující úkol.

	1. Program Task manager je spuštěn a zobrazuje hlavní menu s nabídkou funkcí (1. Přidat úkol, 2. Zobrazit úkoly, 3. Smazat úkol a 4. Ukončit program).
	2. Seznam úkolů není prázdný - seznam již obsahuje několik úkolů.

	1. Vyber funkci "3. Odstranit úkol" stisknutím čísla 3 na klávesnici a potvrď výběr klávesou Enter.

	2. Existující číslo úkolu je zadáno a potvrzeno klávesou Enter. Je zobrazena informace o úspěšném, smazání úkolu: Úkol 'Koupit chleba - celozrnný' byl odstraněn.
